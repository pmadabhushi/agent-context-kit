# Examples

Three examples at increasing complexity. Start with quickstart and work up.

| Example | Complexity | What It Shows |
|---------|-----------|---------------|
| [quickstart/](quickstart/) | Minimal | 4 files for a Todo App. The absolute simplest agent config. Start here. |
| [devops-filled/](devops-filled/) | Medium | Filled-out DevOps templates for a fictional OrderService web app. Shows what a real single-domain config looks like. |
| [greenfield-energy/](greenfield-energy/) | Full | Three personas (DevOps, Coding, Security) for a fictional IoT energy company. Shows how templates adapt to a non-standard domain (physical hardware fleet management). |

## Which One Should I Read?

- Never configured an AI agent before? → [quickstart/](quickstart/)
- Want to see a realistic single-team config? → [devops-filled/](devops-filled/)
- Want to see multi-persona, cross-domain configuration? → [greenfield-energy/](greenfield-energy/)

## How to Use These

1. Browse the example to understand the pattern
2. Go to `templates/` and pick the domain closest to your work
3. Copy the template into your repo
4. Replace `[placeholder]` values using the example as a reference
